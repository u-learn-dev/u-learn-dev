დააინსტალირეთ Node.js Windows-ზე

საინსტალაციო ნაბიჯები

  1. გადმოწერეთ Windows installer  the Nodes.js® ვებსაიტიდან. 

  2. აირჩიეთ LTS version რომელიც მარცხნივ არის ნაჩვენები. 

  3. დარანეთ ინსტალერი  (.msi ფაილი რომელიც წინა ნაბიჯზე გადმოწერეთ.)

  4. მიჰყევით ფრომფთებს ინსტალერში  (დაეთანხმეთ ლიცენზიის შეთანხმებას, დააქლიქეთ NEXT ღილაკს ბევრჯერ და დაეთანხმეთ დეფოლთ საინსტალაციო პარამეტრებს).

 

5. დაარესტარტეთ თქვენი კომპიუტერი. თქვენ ვერ შეძლებთ Node.js® -ის გაშვებას სანამ არ დაარესტარტებთ კომპიუტერს.

6. დარწმუნდით Hyper ტერმინალის გახსნით და "node --version" ბრძანების ჩაწერით(გაშვებით), რომ node-ი წარმატებით დაინსტალირდა: node --version

თქვენ უნდა დაინახოთ node-ს ვერსია, რომელიც ახლახანს დააინსტალირეთ.
