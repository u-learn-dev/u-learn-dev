Node-ის ახალ ვერსიებში პრობლემების გადაჭრა

ზოგიერთ სტუდენტს შესაძლოა Node-ის უფრო ახალი ვერსია ჰქონდეს, ვიდრე ამ ვიდეოებშია. ამ შემთხვევაში, ზოგიერთი ფექიჯი რომელიც არის starting files(სასტარტო ფაილები)-ში, შესაძლოა არ იყოს თავსებადი თქვენი Node-ის ვერსიასთან.


ერთ-ერთი გზა ამ პრობლემის გადასაჭრელად არის npm-check-updates ფექიჯის გამოყენება, რათა განვაახლოთ დეფენდენსიების(იგულისხმება ფექიჯები) ვერსიები საწყის ფაილებში.

https://www.npmjs.com/package/npm-check-updates


პროექტს რომ ამოაარქივებთ, cd-ის რომ იზამთ პროექტის შიგნით, სანამ npm install-ს გაუშვებთ


გაუშვით:

npm install -g npm-check-updates


შემდეგ:

ncu -u


შემდეგ:

npm install


შემდეგ:

npm start
