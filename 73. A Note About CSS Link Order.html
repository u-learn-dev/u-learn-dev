შენიშნვა CSS ლინკების თანმიმდევრობის შესახებ

გამარჯობათ მეგობრებო,

პატარა შენიშვნა. თუ აღმოაჩენთ რომ თქვენი პირადად არჩეული სტილები არ მუშოაბს, დარწმუნდით რომ header-ში სწორადაა link თეგების თანმიმდევრობა.

CSS-ის სტილები აისახება იმის მიხედვით თუ რა თანმიმდევრობით არის ისინი დალინკული თქვენს HTML კოდში.

თუ გქონდათ ორი stylesheet-ი მაგალითად styles1.css და styles2.css რომელიც ორივე იგივე ელემენტს უკავშირდება მაგალითად.

styles1.css

  1. body {
  2. background-color: red;
  3. }

styles2.css

  1. body {
  2. background-color: blue;
  3. }

თუ head სექციის შიგნით ლინკები ამ თანმიმდევრობითაა:

  1. <link rel="stylesheet" href="styles1.css">
  2. <link rel="stylesheet" href="styles2.css">

საბოლოო შედეგის ამსახველი გვერდი იქნება ლურჯი.

მაგრამ თუ თქვენ ასე განალაგეთ ლინკები:

  1. <link rel="stylesheet" href="styles2.css">
  2. <link rel="stylesheet" href="styles1.css">

საბოლოო შედეგის ამსახველი გვერდი იქნება წითელი.

ორივე სტილი აისახება, მაგრამ პრიორიტეტი ბელო სტილს ენიჭება.

ამ ლოგიკის მიხედვით, თუ თქვენი სტილები არ ფარავს ბუთსთრეფის სტილებს, თქვენ მხოლოდ გჭირდებათ რომ თქვენი ლინკი გადაიტანოთ ბუთსთრეფ CDN ლინკის შემდეგ:

  1. <link href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" rel="stylesheet">
  2. <link rel="stylesheet" href="css/styles.css">

ეს ნიშნავს რომ თქვენ პირველად ხსნით ბუთსთრეფ სტილსს და შემდეგ მას გადაფარავთ თქვენი სტილებით.

CSS-სა და JavaScript-სგან გასნხვავებით, HTML-ის კოდი ზევიდან ქვევით სრულდება, ასე რომ თქვენი კოდის თანმიმდევრობა მნიშვნელოვანია.

საუკეთესო სურვილებით,
