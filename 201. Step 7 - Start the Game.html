рЃюрЃљрЃЉрЃўрЃ»рЃў 7 - рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЌрЃљрЃЏрЃљрЃЌрЃў

1. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ jQuery рЃарЃљрЃЌрЃљ рЃЊрЃљрЃљрЃцрЃўрЃЦрЃАрЃўрЃарЃЮрЃЌ рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЎрЃџрЃљрЃЋрЃўрЃљрЃбрЃБрЃарЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃљрЃГрЃћрЃарЃћрЃю, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃћрЃА рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃљ, рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЌ nextSequence().

рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃќрЃљ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ рЃњрЃљрЃАрЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃџрЃљрЃЊ, рЃЊрЃљрЃўрЃгрЃДрЃЮ рЃЌрЃБ рЃљрЃарЃљ рЃЌрЃљрЃЏрЃљрЃерЃў рЃБрЃЎрЃЋрЃћ, рЃљрЃАрЃћ рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃю nextSequence()-рЃА рЃњрЃљрЃЏрЃЮрЃўрЃФрЃљрЃ«рЃћрЃЉрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЏрЃљрЃерЃўрЃю рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ рЃЊрЃљрЃљрЃГрЃћрЃарЃћрЃю рЃдрЃўрЃџрЃљрЃЎрЃА..

2. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ level рЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЊрЃЮрЃюрЃћ 0-рЃЊрЃљрЃю.

3. h1- рЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў рЃўрЃгрЃДрЃћрЃЉрЃљ "Press A Key to Start"- рЃЌрЃЦрЃЏрЃўрЃЌ, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЌрЃљрЃЏрЃљрЃерЃў рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃљ, рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃћрЃА рЃарЃЮрЃЏ рЃЊрЃљрЃћрЃгрЃћрЃарЃЮрЃА "Level 0".

4. nextSequence()-рЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ, рЃњрЃљрЃќрЃќрЃљрЃарЃЊрЃћрЃЌ level 1-рЃўрЃЌ рЃДрЃЮрЃЋрЃћрЃџ рЃ»рЃћрЃарЃќрЃћ рЃарЃЮрЃЊрЃћрЃАрЃљрЃф nextSequence() рЃљрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃФрЃљрЃ«рЃћрЃЉрЃБрЃџрЃў.

5. nextSequence()-рЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ, рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ h1 рЃџрЃћрЃЋрЃћрЃџрЃўрЃА рЃЋрЃћрЃџрЃўрЃБрЃА рЃерЃћрЃфрЃЋрЃџрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.

рЃЌрЃБ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃгрЃљрЃЋрЃљ, рЃљрЃАрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃЮрЃЊрЃћрЃА рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃА рЃЊрЃљрЃљрЃљрЃарЃћрЃцрЃарЃћрЃерЃћрЃЉрЃЌ:

 

рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃЌрЃБ рЃарЃљрЃўрЃЏрЃћ рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ, рЃАрЃфрЃљрЃЊрЃћрЃЌ рЃЦрЃЋрЃћрЃЋрЃўрЃЌ рЃЕрЃљрЃЏрЃЮрЃАрЃЦрЃарЃЮрЃџрЃЋрЃљ. ­ЪЉЄ

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ 1. рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ started to toggle to true рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЌрЃљрЃЏрЃљрЃерЃў рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃљ рЃарЃЮрЃЏ рЃњрЃљрЃўрЃњрЃЮрЃЌ рЃЏрЃљрЃарЃЌрЃљрЃџрЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЏрЃЮрЃЏрЃЊрЃћрЃЋрЃюрЃЮ рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃЏрЃљ рЃљрЃа рЃБрЃюрЃЊрЃљ рЃљрЃљрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃЮрЃю nextSequence().

рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ 3. h1-рЃА рЃљрЃЦрЃЋрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃБрЃџрЃў level-title-рЃўрЃА id рЃарЃЮрЃЏрЃџрЃўрЃА рЃљрЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃф jQuery-рЃўрЃЌ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ.

рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ 5.  рЃЌрЃЦрЃЋрЃћрЃю рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ jQuery-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃћрЃарЃЌрЃ«рЃћрЃџ, рЃарЃљрЃЌрЃљ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃЌ h1 -рЃўрЃА рЃљрЃўрЃЊрЃў рЃЏрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃўрЃЌ: level-title.
