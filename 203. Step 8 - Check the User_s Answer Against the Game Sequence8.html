ნაბიჯი 8 - შეადარეთ მომხმარებლის პასუხი Game Sequence8-ს.

უკვე, მგონი ღირს იმის ნახვა როგორ მუშაობს Simon game-ი.

პირველ რიგში, თამაში გვიჩვენებს პირველ ფერს მიმდევრობაში (ლურჯი). მომხმარებელი აქლიქებს ლურჯ ბათონზე.

შემდეგ, თამაში გვიჩვენებს შემდეგ ფერს (წითელი), მომხმარებელმა უნდა დაიმახსოვროს რომ მიმდევრობა არის ლურჯი, წითელი და ასე შემდეგ და ასე შემდეგ.

თუ მომხმარებელს აერევა თანმიმდევრობა, მაშინ თამაში დასრულდება.

თქვენ შეგიძლიათ თქვენით სცადოთ კოდის დაწერა და გაიგოთ თუ როგორ უნდა მიაღწიოთ ამ ლოგიკას ან შეგიძლიათ მიჰყვეთ ქვემოთ მოცემულ გამოწვევის ნაბიჯებს:

1. შექმენით ახალი ფუნქცია სახელად, checkAnswer(), რომელმაც უნდა მიიღოს ერთი უნფუთი სახელად currentLevel

2. გამოიძახეთ checkAnswer() იმის შემდეგ რაც მომხმარებელი დააქლიქებს და აირჩევს თავის პასუხს, და გადაეცემა ბოლო პასუხის ინდექსი მომხმარებლის თანმიმდევრობაში.

მაგალითად, თუ მომხმარებელი დააქლიქებს red, green, red, yellow,-ს ბოლო პასუხის ინდექსი არის 3.

3. დაწერეთ if statementcheckAnswer()-ის შიგნით, რატა შეამოწმოთ ბოლო მომხმარებლის პასუხი თუ შეესაბამება თამაშის ნიმუშს. თუ კი, მაშინ log "success", თუ არა და log "wrong".

თქვენ ახლა შეგიძლიათ გამოიყენოთ ეს ლოგ-ის წესები userClickedPattern და gamePattern ის ველიუების შესვლასთან ერთად და ქრომის დეველოპერის თულსის კონსოლში რათა შეამოწმოთ მუშაობს თუ არა თქვენი კოდი ისე როგორც ელოდებოდით და გაუკეთეთ debug-ი თქვენს კოდს, როგორც დაგჭირდებათ. როდესაც დააასრულებთ, თავისუფლად შეგიძლიათ ეს ლოგის წესები მოაშოროთ.

4. თუ მომხმარებელმა შეიყვანა ყველაზე ბოლო პასუხი სწორად მესამე საფეხურზე, მაშინ შეამოწმეთ თუ დაასრულეს თავიიანთი მიმდევრობა კიდევ ერთი if statement-ით.

5. გამოიძახეთ nextSequence() 1000 მილიწამი დაგვიანების შემდეგ.

6. როდესაც nextSequence() გამოიძახება, დაარესეთეთ userClickedPattern ცარიელი array-ით რათა, შემდეგი დონისთვის მზად იყოს.

 

ახლა უნდა შეგეძლოთ თამაში.
