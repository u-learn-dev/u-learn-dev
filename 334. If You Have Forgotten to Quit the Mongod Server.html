თუ დაგავიწყდა Mongod Server-ის გათიშვა

შეგიძლია დააწვე CTRL + C შენს ტერმინალში რათა გათიშო mongodb-სთან კავშირი.


თუ გათიშე Terminal ან Hyper და დაგავიწყდა mongodb-სთან კავშირის გათიშვა, შესაძლოა შეგხვდეს ეს ერორი: 

  1. 2018-11-04T16:17:53.473+1300 E STORAGE  [initandlisten] Failed to set up listener: SocketException: Address already in use
  2. 2018-11-04T16:17:53.474+1300 I CONTROL  [initandlisten] now exiting
  3. 2018-11-04T16:17:53.474+1300 I CONTROL  [initandlisten] shutting down with code:48

ამ შემთხვევაში, მიყევი ამ ნაბიჯებს, რათა გათიშო mongod პროცესი:

1. გახსენი ახალი Hyper ტერმინალი

2. ჩასვი ეს ბრძანება ტერმინალში:

sudo pkill -f mongod

3. შეიყვანე პაროლი რომელსაც შენ მაკში ლოგინისას იყენებ.

4. გახსენი ახალი Hyper ტერმინალი. ახლა კვლავ უნდა შეგეძლოს mongod ბრძანების გაშვება.
