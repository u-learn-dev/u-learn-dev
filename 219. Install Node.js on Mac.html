Node.js-ის დაინსტალირება Mac-ზე

საინსტალაციო ნაბიჯები

  1. გადმოწერეთ Mac installer-ი  the Nodes.js® ვებსაიტიდან. 

  2. ამოირჩიეთ LTS ვერსია რომელიც მარცხნივ არის ნაჩვენები. 

  3. გაუშვით ინსტალერი (ორჯერ დააჭირეთ .pkg ფაილს, რომელიც წინა ნაბჯის მიხედვით გადმოწერეთ.)

    შენიშვნა: თუ macOS Catalina 10.15 გაქვთ, შესაძლოა გაფრთხილება ამოგიგდოთ, რომელიც ასე გამოიყურება:

    ეს პრობლემა რომ მოგვარდეს, მარჯვენა ღილაკით დააქლიეთ ( control ღილაკს დააწექით კლავიატურაზე და დააქლიქეთ) .pkg ფაილს და აირჩიეთ "Open With" და შემდეგ აირჩიეთ "Installer (default)".

     

  4. მიჰყევით ფრომფთებს ინსტალერში  ( დაეთანხმეთ ლიცენზიის შეთანხმებას, დააქლიქეთ NEXT ღილაკს ბევრჯერ და დაეთანხმეთ დეფოლთ საინსტალაციო პარამეტრებს).

  5. დარწმუნდით Hyper ტერმინალის გახსნით და "node --version" ბრძანების გაშვებით, რომ node-ი წარმატებით დაინსტალირდა.

    node --version

თქვენ უნდა დაინახოთ node-ს ვერსია რომელიც ახლახანს დააინსტალირეთ.
