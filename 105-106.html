Javascript-ის ცვლადების სავარჯიშო

მოცემულია ქვემოთ არსებული კოდი: შეგიძლიათ დაწეროთ კოდი რომელით a-ს და b-ს ველიუებს(მნიშვნელობებს) შეუცვლის?

  1. var a = "3";
  2. var b = "8";

ცვლადი a-ს ველიუ არის "8".

და ცვლადი b-ს ველიუ არის "3".

როდესაც კოდი გაეშვება, აუთფუთი უნდა იყოს:

a არის 8

b არის 3

არ შეცვალოთ არაფერი მოცემულ კოდში.

არ  გაქვთ უფლება რომ a-ს და b-ს პირდაპირ რიცხვები მიანიჭოთ.

არ უნდა გამოაცხადოთ(შექმნათ) a და b ცვლდაები თავიდან.

მინიშნება: გამოიყენეთ ეს code playground რომ გაუშვათ კოდი და ნახოთ თუ შეესაბამება თქვენს მოლოდინს.

მინიშნება: ამის გამოსავალი მხოლოდ კოდის 3 ხაზია.
