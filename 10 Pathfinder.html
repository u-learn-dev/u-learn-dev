იმსათვის რომ კურსი თქვენ მოგარგოთ და საუკეთესო სწავლის გამოცდილება მიიღოთ, გთხოვთ აირჩიოთ გამოცდილების დონე, რომელიც ყველაზე კარგად გახასიათებთ თქვენ:

1. ნული გამოცდილება: მე სრულიად დამწყები ვარ პროგრამირებაში.

2. ცოტა გამოცდილება: მე მაქვს ნასწავლი ცოტა html და css წარსულში.

3. გარკვეულწილად გამოცდილი: მე მაქვს 6 თვეზე ნაკლები გამოცდილება ჯავასკრიპტსა და ვებ დეველოპმენთში.

4. გამოცდილი დეველოპერი, ვებში დამწყები: მაქვს დიდი გამოცდილება პროგრამირებაში მაგრამ ვებ დეველოპერობაში ახალი ვარ.

5. გამოცდილი ვებ დეველოპერი: მინდა შევისწავლო ყველაზე ახალი ჯავასკრიპტის ფრეიმვორქები.

6. მივდა შევისწავლო რეაქტი:ვარ გამოცდილი ვებ დეველოპერი მაგრამ მინდა ვისწავლო რეაქტი.

 

მოძებნეთ რეკომენდირებული გზა თქვენთვის:

 

1. ნული გამოცდილება

სრუიად დამწყებებისთვის, მე გირჩევთ რომ დაიწყოთ სულ თავიდან და დარწმუნდით, რომ არ ტოვებთ რომელიმე მოდულს ან გაკვეთილს. ტუტორიალები და გამოწვევები ერთმანეთზე იგება, ამიტომ კურსის გავლა მთაზე ასვლას გავს. ვიცი ზოგიერთი მოდულები ძალიან მაგარია და არსებობს ცდუნება რომ პირდაპირ იმაზე გადახტეთ, მაგრამ სწორი საწყისი საკითხების გარეშე, ადვილია დაკარგვა და მოტივაციის გაქრობა.

დარწმუნდით, რომ სრულიად კომფორტულად ხართ თითოეული მოდულის ცონსეპტებთან განრიგის შესაბამისად, სანამ შემდეგზე გადახვალთ. ზოგადად, დამწყებებისთვის ჩემი რეკომენდაციაა ერთი კვირის დახარვა თითოეულ მოდულზე.

 

2. ცოტა გამოცდილება

რაღაც გამოცდილება გაქვთ, ამიტომ ვცადოთ და შევინარჩუნოთ ეს სუფთად და არ დავხარჯოთ დრო, იმის გამეორებაზე რაც უკვე იცით. მე გირჩევთ შუალედური დონის ტუტორიალების დაწყებას და აუცილებლად შეასრულეთ ყველა გამოწვევა. თუ გამოწვევა მიგაჩნიათ ძალიან რთული, მაშინ შესაძლოა ღირდეს წინა ტუტორიალებზე დაბრუნება და კონსეპტების გადახედვა.

დაიწყეთ მე-3 სექციიდან: "საშუალო დონე HTML-ში"

 

3. გარკვეულწილად გამოცდილი

პროგრამირების კონცეფციები თქვენთვის მარტივი უნდა იყოს, განსაკუთრებით მაშნ თუ უკვე ბევრი რამ იცით ჯავასკრიპტის შესახებ. მე გირჩევთ პროგრამირების შესავლის მოდულების გამოტოვებას და მათ განხილვას მხოლოდ იმ შემთხვევაში თუ დაგჭირდებათ. ამის ნაცვლად, სცადეთ ყველა გაკვეთილის დასრულება, რომელსაც სიტყვა გამოწვევა აწერია სათაურად, რათა დარწმუნდეთ რომ კონცეფციები კარგად გესმით და უსაფრთხოდ შეგიძლიათ გამოტოვოთ ისინი.

დაიწყეთ მე-13 სექციიდან: "მაღალი დონე ჯავასკრიპტში - DOM-ში მანიპულირება"

 

4. გამოცდილი დეველოპერი, ვებში დამწყები

თქვენ ალბათ სწრაფად გინდათ რომ დაეწიოთ ვებ დეველოპმენტს. ხალხის უმეტესობას ამ კემპში ექნება რაღაც გამოცდილება html და css -სთან, ამიტომ გირჩევთ რომ დაიწყოთ ჯავასკრიპტის მოდულებით. რადგან ჯავასკრიპტის ბევრი უცნაური თვისება აქვს, შესაძლებელია ცოტა შოკისმომგვრელი იყოს ზოგიერთი ადამიანისთვის, რომლებმაც უფრო უსაფრთხო ენები იცის (მაგალითად სვიფტი).

დაიწყეთ მე-9 სექციიდან: "ჯავასკრიპტის შესავალი - ES6"

 

5. გამოცდილი ვებ დეველოპერი

თქვენ იცით ჯავასკრიპტი, html და css, ისე როგორც თქვენი ხელის მეორე მხარე, მაგრამ ყოველ კვირა გესმით ახალი ჯავასკრიპტის ფრეიმვორკის შესახებ და გადაწყვიტეთ რომ დროა რომ საქმეში ჩებათ და გაიგოთ რაზე ლაპარაკობენ ეს ბავშვები. ჩვენ დავიწყებთ ნოუდით და გადავალთ ექსპრესზე, იჯიეს-ზე, მონგოზე და კიდევ უფრო მეტზე, რათა სწრაფად აითვისოთ (და იმედია php-საც მოიშორებთ) .

დაიწყეთ მე-18 სექციიდან: "Node.js (ნოუდ ჯეი ესი)"
