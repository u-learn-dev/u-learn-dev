Hyper Terminal-ის დაინსტალირება

შემეგ გაკვეთილზე, ჩვენ დავაინსტალირებთ Hyper Terminal-ს. როგორც ყოველთვის ინსტალაციისთვის ბევრი რამ არის რაც შესაძლოა არასწორად წავიდეს. უბრალოდ უკან გადმოდით თუ Hyper-ის ინსტალაციისას რაიმე პრობლემას წააწყდებით.

თუ ხართ macOS Catalina-ზე, Hyper -ს აქვს unzipping-ის დროს პრობლემები. უბრალოდ გადმოწერეთ "The Unarchiver" და გამოყენეთ ეს რათა გადმოწერილ ფაილს unzip გაუკეთოთ..

1) გადმოწერეთ the unarchiver ამ ვებსაიტიდან : https://theunarchiver.com/

2)მარჯვენა ღილაკით დააჭირეთ გადმოწერილ Hyper ზიპ ფაილს და ამოირჩიეთ open with Unarchiver.

 

თუ Windows-ზე ხართ და Hyper არ მუშაობს ან ინსტალაცია ფეილდება , დარწმუნდით რომ გადმოწერილი და დაინსტალირებული გაქვთ Git Bash.

გადადით აქ: https://git-scm.com/downloads

და თქვენ განაახლეთ კონფიგურაციის ფაილი, როგორც ვიდეოშია მოცემული..

თუ ამის გაკეთების შემდეგ, კიდევ არ მუშაობს ის, შეგიძლიათ სცადოთ Git Bash Terminal-ის გამოყენება Hyper-ის ნაცვლად.

1) გადმოწერეთ Git Bash აქედან: https://gitforwindows.org/

2) შეგიძლიათ მონიშნოთ ეს ბოქსი, რათა Desktop icon დაემატოს

installation wizard-ში

3) აირჩიეთ Use Vim

4) გამოიყენეთ OpenSSH

5) გამოიყენეთ OpenSSL

6) ნახეთ Windows-style

7) გამოიყენეთ MinTTY

8) Enable File System cashing და the Credential Manager

9) დატოვეთ the experimental features მოუნიშნავი და დააქლიქეთ install-ს.

 

10)როდესაც მორჩება, ნახეთ Launch Git Bash ოფცია და დააქლიქეთ Finish-ს.

11) ახლა the Git Bash Terminal ამუშავდება და შეგიძლიათ Hyper-ის ნაცვლად გამოიყენოთ.
