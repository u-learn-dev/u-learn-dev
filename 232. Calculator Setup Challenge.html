კალკულატორის setup-ის გამოწვევა

სანამ ჩვენ შევქმნით კალკულატორის ვებსაიტს, ჩვენ დაგვჭირდება რომ ახალი პროექტი ავაწყოთ. მიჰყევით ქვევით მოცემულ ნაბიებს Hyper Terminal-ის გამოყენებით, რათა ეს გამოწვევა შეასრულოთ:

  • შექმენით ახალი საქაღალდე თქვენს დესკტოპზე სახელად Calculator

  • შედი ამ საქაღალდეში

  • Calculator საქაღალდის შიგნით, შექმენით ახალი ფაილი სახელად calculator.js

  • შექმენით ახალი NPM package-ი

  • გახსენით პროქეტის საქაღალდე ატომში 

  • NPM- ის გამოყენებით დააინსტალირეთ express მოდული

  • მოითხოვეთ express-ი ჩვენს calculator.js-ში

  • დაასეთაფე express-ი

  • შექმენით root route get method app.get()-ით

  • გააგზავნეთ სიტყვები Hello World root route-დან საპასუხოდ

  • გაუშვი სერვერი 3000 პორტზე app.listen- ით

  • დასტარტე სერვერი nodemon-ით
