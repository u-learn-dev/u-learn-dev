Nodemon-ის ინსტალაცია

თუ პრობლემებს წააწყდებით nodemon-თან შემდეგ გაკვეთილზე, დაბრუნდით და სცადეთ ეს გამოსავალი გზები.

 

გამარჯობათ მეგობრებო,

შემდეგ გაკვეთილზე, მე აგიხსნით თუ როგორ უნდა დააინსტალიროთ nodemon, რომელიც არის npm package, რომელიც უფრო გაგვიადვილებს ჩვენი სერვერების ავტომატურად დასტარტვას.

nodemon-ის ინსტალაციის პრობლემები

შემდეგ გაკვეთილზე, განახებთ ოფიციალურ საინსტალაციო გვერდს nodemon-ისგან, მაგრამ დაგჭირდებათ ადმინის უფლებები რათა თქვენს სისტემაზე დაინსტალირდეს.

ასე რომ, როდესაც ეს ბრძანება არ მუშაობს:

  1. npm install -g nodemon

გირჩევთ ამის გამოყენებას, მის ნაცვლად:

  1. sudo npm install -g nodemon

ის შემდეგ მოგთხოვთ რომ პაროლი შეიყვანოთ, რომელსაც კომპიუტერში შესვლის დროს იყენებთ.

ასე რომ ამ ეტაპზე თუ რაიმე პრობლემა გაქვთ.

1. დარწმუნდით რომ კომპიუტერში ადმინის პრივილეგიების მქომე account-ით ხართ შესულნი.

2. გამოიყენეთ sudo საინსტალაციო ბრძანების წინ.

3. როდესაც მოგთხოვთ პაროლი შეიყვანეთ. შენიშნავთ რომ პაროლს როდესაც წერთ, ის არ ჩანს. უბრალოდ ენთერს დააჭირეთ როდესაც მთლიანად ჩაწერთ მას.
